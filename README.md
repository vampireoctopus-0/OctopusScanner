# VampireScan v1.0 🩸

A lightweight, multi-threaded Python Port Scanner and Banner Grabber designed for quick network reconnaissance during penetration testing engagements.

## 🚀 Features
- **Fast Port Scanning:** Built using Python's native socket library.
- **Error Handling:** Implements timeouts to prevent hang-ups on filtered ports.
- **Zero Dependencies:** Runs entirely on standard Python libraries (No external installation required).
